# 更新记录

本项目遵循[语义化版本](https://semver.org/lang/zh-CN/)。

## 0.3.0 - 待发布

- 采用 `cmd/yuxin` 与 `internal/app` 的标准 Go CLI 布局，保持单一应用包。
- 新增固定宽度纯文本分享卡片，默认使用合成数据，并提供工作日倒计时卡。
- 新增只监听本机的轻量浏览器入口，使用一次性启动密钥和真实数据二次确认；Release 包提供各平台非终端启动文件。
- 新增金额、退休年月字段级隐藏，以及退休轻量模式和年/月/日/工作日单位切换。
- 新增配置导出、校验导入和确认清除。
- 新增 `update --force`；当前年份缺少随包节假日数据时，每次启动都会提醒强制刷新。
- 修复部分 shell 环境把安装脚本变量与相邻多字节字符错误解析的问题，并增加真实安装测试。

## 0.2.0 - 2026-07-16

- 新增 `yuxin update`，可下载并校验 GitHub 最新正式版。
- 新增 `s` 隐私演示模式，使用固定合成数据生成可分享截图。
- 退休进度改为按 18 岁起算的统一阶段进度，无需收集参加工作年份。
- 首次启动默认关闭退休和资产模块，聚焦今日入账与节假日倒计时。
- Release 产物使用稳定文件名，README 下载链接始终指向最新版。
- 提供自动识别平台的 Shell 和 PowerShell 一行安装入口。

## 0.1.0 - 2026-07-16

- 使用无第三方依赖的 Go 原生单文件实现，快速启动且无需安装运行时。
- 提供每秒刷新的工资、工时和下班倒计时终端仪表盘。
- 提供法定退休时间、退休进度和“现在退休每天可花”估算。
- 支持从仪表盘按 `e` 分模块编辑薪资、工作时间、退休和多账户资产。
- 支持应急保留金、金额缩写以及身份证出生日期和性别解析。
- 内置 2026 年节假日数据和假期进度可视化，全程无需联网。
- 提供 macOS、Windows、Linux 单文件发行方式。
