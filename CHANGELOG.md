# 更新记录

本项目遵循[语义化版本](https://semver.org/lang/zh-CN/)。

## Unreleased

- 新增 `yuxin share --anonymous`，在本地预览降精度字段与区间，输入 `SHARE` 后才连接公开数据服务提交。
- 新增余薪匿名公开数据看板，展示经隐私门槛处理的聚合数据和审核通过的匿名回声。
- 薪资模式增加年薪，切换月薪、日薪、时薪或年薪时会按当前工时等价换算整数默认值。
- 存款目标增加“心愿物品”模式，用实时存款显示目标进度和还差金额。

## 0.6.0 - 2026-07-17

- 重新设计纯文本分享卡片与演示画面，演示内容改由当前 Go 渲染逻辑按秒重算，并补充 GIF、宣传视频自动合成、本地拟人配音音轨、配音文案和旧渲染器归档。
- 实时存款余额会从存款起算日起，按当前薪资与工时配置回算完整工作日工资，并继续实时加入今日工资；可用年份的节假日和调休补班沿用工作日历口径。
- Release 新增 Windows ARM64 与 Linux ARM64，并让安装脚本和内置更新器自动选择对应架构。
- 新增 Homebrew Formula 生成与 Tap 自动更新流程，发布后可同步 macOS Apple Silicon 和 Intel 安装包。
- 直接解压 ZIP 运行时会自动读写可执行文件同目录的 `yuxin.toml`，作为可随目录移动的便携配置。
- “目标每月可花”默认值调整为 3,000 元，存款与退休模块仍默认关闭。
- 退休和存款卡片改为同高的平行口径，补充 emoji 标题与节制的语义配色，将“预算”统一为更轻量的“可花”，并移除重复的年度换算。

## 0.5.0 - 2026-07-17

- 存款预算文案统一为“撑到退休”，完整面板标题新增按每日可花金额变化的生活体感彩蛋。
- 节假日数据改为自动发现随包年份；Release ZIP 只包含最新年度文件，缺少当年数据时提醒检查普通更新。
- 隐私和视图快捷键持续显示当前档位，非工作状态使用对应提示文案；旧版退休隐私配置自动映射为全部隐藏。
- Windows 更新与卸载脚本采用有限重试，并在确认删除程序后才清理用户 `PATH`。
- 增加真实 CLI 子进程、Unix 中断、Windows 脚本与 fuzz 测试；CI 增加 ShellCheck、actionlint 和固定版本的 GitHub Actions。
- 配置增加金额与工作天数上限，节假日、退休、渲染格式和配置校验按职责拆分，长期工作日统计改为整周计算。

## 0.4.0 - 2026-07-17

- 产品重新聚焦终端使用，移除 `web` 命令和 Release 中的桌面启动壳。
- 退休配置收口为年龄/出生年月与性别，分行显示整数剩余总年数、总月数、总天数和进度条；性别默认男。
- 隐私模式与配置、退出同级，仪表盘按 `p` 直接切换并保存三个档位；删除退休单位切换快捷键。
- 存款保留单一余额，新增实时余额、立即退休后的日月年预算，以及目标月支出的存款进度和还差金额。
- 状态栏新增可自定义口号，并简化节假日进度文案。
- 修正上班前的倒计时口径：尚未上班时显示距离上班，下班后只显示已经下班。
- 午休配置收口为一个分钟数，程序自动推导开始和结束时间。
- 删除主仪表盘中重复的 `?` 帮助面板，快捷键直接保留在底部。
- 合并演示与计算详情快捷键，按 `v` 循环三种视图。
- 新增 `yuxin uninstall`；默认保留配置，`--purge` 可确认后同时清除配置。
- 新增基于当前 Go 渲染内容的终端演示生成脚本，并在 README 嵌入 GIF。

## 0.3.0 - 2026-07-16

- 采用 `cmd/yuxin` 与 `internal/app` 的标准 Go CLI 布局，保持单一应用包。
- 新增固定宽度纯文本分享卡片，默认使用合成数据，并提供工作日倒计时卡。
- 新增只监听本机的轻量浏览器入口，使用一次性启动密钥和真实数据二次确认；Release 包提供各平台非终端启动文件。
- 新增金额、退休年月字段级隐藏，以及退休轻量模式和年/月/日/工作日单位切换。
- 新增配置导出、校验导入和确认清除。
- 新增 `update --force`；当前年份缺少随包节假日数据时，每次启动都会提醒强制刷新。
- 修复部分 shell 环境把安装脚本变量与相邻多字节字符错误解析的问题，并增加真实安装测试。

## 0.2.0 - 2026-07-16

- 新增 `yuxin update`，可下载并校验 GitHub 最新正式版。
- 新增 `s` 隐私演示模式，使用固定合成数据生成可分享截图。
- 退休进度改为按 18 岁起算的统一阶段进度，无需收集参加工作年份。
- 首次启动默认关闭退休和资产模块，聚焦今日入账与节假日倒计时。
- Release 产物使用稳定文件名，README 下载链接始终指向最新版。
- 提供自动识别平台的 Shell 和 PowerShell 一行安装入口。

## 0.1.0 - 2026-07-16

- 使用无第三方依赖的 Go 原生单文件实现，快速启动且无需安装运行时。
- 提供每秒刷新的工资、工时和下班倒计时终端仪表盘。
- 提供法定退休时间、退休进度和“现在退休每天可花”估算。
- 支持从仪表盘按 `e` 分模块编辑薪资、工作时间、退休和多账户资产。
- 支持应急保留金、金额缩写以及身份证出生日期和性别解析。
- 内置 2026 年节假日数据和假期进度可视化，全程无需联网。
- 提供 macOS、Windows、Linux 单文件发行方式。
