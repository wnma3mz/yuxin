# Yuxin CLI 配置逻辑

本文只记录当前 CLI 可见的配置入口、交互顺序和保存规则。网页版的浏览器本地配置见 [`web/README.md`](../web/README.md)。

## 配置入口

- 首次启动会自动创建默认配置，不强制进入向导。
- 仪表盘按 `e` 或运行 `yuxin config` 进入同一套菜单。
- `yuxin config export FILE` 导出配置。
- `yuxin config import FILE` 校验后覆盖当前配置。
- `yuxin config clear` 确认后删除当前配置。
- `--interval` 只覆盖本次运行，不写入文件。

配置路径按以下优先级确定：

1. `--config FILE`
2. `YUXIN_CONFIG`
3. 可执行文件同目录的 `yuxin.toml`（ZIP 便携模式）
4. `$XDG_CONFIG_HOME/yuxin/config.toml`
5. `~/.config/yuxin/config.toml`

## 当前主菜单

```text
当前配置
  1 今日入账   月薪 ¥8,000.00
  2 工作时间   09:00–18:00，周 1,2,3,4,5
  3 退休倒计时 关闭
  4 存款       关闭
  5 目标计划   关闭
  6 更多设置   刷新 1s
  7 匿名贡献   预览后确认上传
选择要修改的部分 [0]:
```

输入 `1`–`7` 执行对应操作；输入 `0` 或直接回车结束。配置只在整体校验通过且内容确实变化时原子保存。

## 配置项

### 1. 今日入账

1. 选择月薪、日薪、时薪或年薪。
2. 输入对应金额。
3. 月薪模式补充每月工作天数。

金额必须大于 `0` 且不超过 `1,000,000,000,000`，支持 `20w`、`20万`、`200k` 和 `200,000`。切换薪资模式时，默认值会按当前工时等价换算并取整。

### 2. 工作时间

1. 工作日，`1` 表示周一，`7` 表示周日。
2. 上班时间和下班时间，使用 `HH:MM`。
3. 午休分钟数，`0` 表示不扣除。

至少选择一个工作日；下班必须晚于上班；午休必须小于当日总工时。

### 3. 退休倒计时

```text
退休倒计时：0 关闭 / 1 按年龄和性别估算 / 2 手动填写剩余年数 [0]:
```

- 自动模式只输入当前整数年龄和性别，性别默认男。程序以当天月日反推估算生日，再按延迟退休政策估算。
- 手动模式直接输入距离预计退休的剩余整数年数，不询问年龄和性别。
- 自动模式的进度按 18 岁起算；手动模式无法推断过去人生阶段，因此从设置当天的 `0%` 按剩余期限向前推进。
- 面板同时显示剩余总年数、总月数、总天数和退休进度。

### 4. 存款

只输入一个存款金额；输入 `0` 关闭。实时存款为手动存款加上从起算日至今按当前薪资和工时回算的工资。修改存款金额会从当天重新起算。

退休开启时，面板额外显示“用当前存款撑到预计退休”的每日和每月可花金额。

### 5. 目标计划

```text
目标计划：0 关闭 / 1 躺平月支出 / 2 心愿物品 [0]:
```

- 躺平目标只配置“目标每月可花”，默认 `3,000` 元；需要同时开启存款和退休倒计时。
- 躺平目标用当前存款与“撑到退休”所需总额计算进度，不引入收益率、通胀或养老金假设。
- 心愿目标只输入名称和一次性金额，不需要存款或退休配置；从设置当天开始按当前工资实时推进。
- 两种目标互斥，隐藏金额时同时隐藏目标名称、金额、差额和进度。

### 6. 更多设置

- 刷新间隔：`0.1`–`3600` 秒。
- 自定义口号：最长 40 个字，默认“摸鱼有数，下班有期。”

### 7. 匿名贡献

先在本地预览折算月薪、净工时、每周工作天数和所在区间。存款、退休年数和匿名回声都是选填。只有输入大写 `SHARE` 后才会连接 Supabase；取消或输入结束不会发送请求。

## 隐私快捷键

隐私不放在配置菜单内。仪表盘按 `p` 循环并立即保存：

```text
关闭 → 隐藏金额和存款 → 同时隐藏退休 → 关闭
```

隐私只影响显示，不删除配置。

## 导入、导出与清除

- 导出会提示文件可能包含薪资、存款和退休信息。
- 导入先完整解析和校验，成功后才原子替换；不自动备份。
- 清除必须准确输入 `DELETE`，只删除当前配置文件。
- Unix 下配置目录权限为 `0700`，文件权限为 `0600`；写入时使用同目录临时文件后原子替换。

## 卸载

- `yuxin uninstall` 需输入 `UNINSTALL`，删除当前程序并保留配置。
- `yuxin uninstall --purge` 需输入 `PURGE`，同时删除当前配置。
