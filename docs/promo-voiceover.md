# 余薪宣传片配音文案

适配 20 秒横版宣传片。语气自然、带一点打工人式自嘲，像同事间分享一个顺手的小工具，不使用播音腔。

| 时间 | 画面 | 旁白 |
| --- | --- | --- |
| 0–3.4 秒 | 情绪引入 | 这班不能白上，至少得知道赚了多少。 |
| 3.4–12.4 秒 | 完整仪表盘 | 今天赚了多少、还有多久下班、今年还要上多少天班，余薪都替你算好了。日盼下班，终盼退休。 |
| 12.4–15.5 秒 | 分享卡 | 想晒进度，一键生成分享卡。 |
| 15.5–20 秒 | 品牌收尾 | 余薪，摸鱼有数，下班有期。 |

## 音轨与合成

- 当前已评审配音保存在 `docs/assets/yuxin-promo-voice.m4a`，包含与四个分镜对齐的停顿，默认无需联网重新合成。
- `scripts/render-promo.sh` 会自动把该音轨混入成片；设置 `YUXIN_PROMO_SILENT=1` 生成无声版。
- 如果需要替换人声，先生成同样 20 秒、已完成分镜对齐的音轨，再通过 `YUXIN_PROMO_VOICE_TRACK=/path/to/voice.m4a` 传入。
- 替换声线仍建议使用普通话年轻自然声线，每句单独生成，句尾自然收住。
