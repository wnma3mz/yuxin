# Yuxin 产品与发布路线

## 目标

当前路线聚焦最新版获取、隐私安全的功能展示、非终端启动入口，以及更直观的退休进度。保持单一 Go CLI 和标准库实现。

## 实施项

- [x] 将 GitHub 社区文件收纳到 `.github/`，不引入 `src/` 目录。
- [x] 采用最小标准 Go 布局：入口放入 `cmd/yuxin/`，应用代码和同包测试放入 `internal/app/`。
- [x] 新增 `yuxin update`，从 GitHub 最新正式 Release 下载当前平台的 CLI。
- [x] 新增 `yuxin update --force`；当前年份缺少节假日数据时每次启动提醒刷新 Latest Release。
- [x] 下载后校验 Release 中的 SHA-256，再替换当前程序。
- [x] Release 产物使用稳定文件名，README 通过 `/releases/latest/download/...` 直达最新 ZIP。
- [x] 提供 Shell 和 PowerShell 安装脚本，在终端自动识别平台并安装 Latest Release。
- [x] 新增 `s` 快捷键，使用一套固定合成数据切换隐私演示模式。
- [x] 退休进度改为按 18 岁起计的统一阶段进度，不收集参加工作年份。
- [x] 首次启动默认关闭退休和资产模块，首页聚焦今日入账和节假日倒计时。
- [x] 更新 README 看板、下载入口、命令和数据口径。
- [x] 运行格式化、单元测试、竞态测试、`go vet`、跨平台编译和 Release 产物校验。
- [x] CI 增加覆盖率门槛，全项目语句覆盖率不得低于 78%。
- [x] 完成后续路线验证，发布 `v0.3.0` 并由 GitHub 标记为 Latest Release。

## 已确定的后续路线

### 非终端启动入口

- [x] 实现 `yuxin web` 轻量模式：只作为非终端用户的启动入口，不发展成第二套独立产品界面。
- [x] 只监听本机回环地址，自动打开系统浏览器，复用现有计算和配置逻辑。
- [x] 在 Release ZIP 中提供 macOS `.app`、Windows `.vbs` 和 Linux `.desktop` 轻量启动壳，不引入桌面 GUI 运行时。
- [x] 保持无第三方运行时，验证退出、端口冲突、浏览器启动失败和本地隐私边界。

### 纯文本分享卡片与图库

- [x] 新增一键生成固定宽度、纯文本、纯间距排版的可分享卡片。
- [x] 默认只使用合成数据；只有用户显式选择后才生成真实数据卡片，并在生成前提示隐私风险。
- [x] 提供完整功能演示卡，突出“一键生成可分享画面”。
- [x] 提供工作日倒计时卡，突出“无账号、离线运行、数据只在本地”。

### 退休轻量模式

- [x] 提供“只显示距离退休”的轻量模式，不展示出生日期或法定年龄。
- [x] 允许在年、月、日和工作日口径之间切换。

### 数据与隐私

- [x] 增加配置导出和导入，在导出前明确标示可能含有的敏感字段。
- [x] 支持字段级别的展示隐藏，至少包含金额和退休年月。
- [x] 增加“一键清除本地数据”，并在删除前明确确认目标路径。

## 同类产品调研摘要

本轮调研只用于建立讨论清单，不自动扩大已确认的开发范围。

- [WTFUtil](https://github.com/wtfutil/wtf) 采用可配置模块、独立刷新频率、键盘焦点和主题，说明“按需开启模块”比把所有指标堆在默认首屏更易扩展。
- [Timewarrior](https://timewarrior.net/docs/) 提供日、周、月报告，标签、工作周、节假日、图表和数据导出，说明历史趋势与可导出数据是时间类产品的常见扩展方向。
- [Retirement Countdown](https://apps.apple.com/us/app/retirement-countdown/id1645215938) 提供全屏倒计时、天/周/月/秒单位、桌面小组件和分享卡片，说明“一键生成可分享画面”比让用户自行截取真实数据更安全。
- [RetirementTracker](https://play.google.com/store/apps/details?id=com.rententracker.app) 强调工作日倒计时、无账号、离线运行和本地数据，与 Yuxin 的本地优先方向一致。

## 暂不考虑

- macOS 菜单栏、Windows 托盘、桌面小组件、主题和自定义模块排序。
- 本周/本月收入、发薪日、打点、加班、请假、历史趋势和提醒。
- 多地区节假日、系统日历和独立节假日更新。
- 银行同步、投资、收益预测和财务建议。

## 发布产物

GitHub Release 固定提供以下文件，以便最新版链接和自动更新器长期有效：

```text
yuxin-macos-arm64.zip
yuxin-macos-x86_64.zip
yuxin-windows-x86_64.zip
yuxin-linux-x86_64.zip
```

每个 ZIP 同时发布对应的 `.sha256` 文件。
