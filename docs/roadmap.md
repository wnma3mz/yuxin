# Yuxin 产品与发布路线

## 目标

当前路线聚焦终端内的今日入账、下班倒计时、节假日进度和隐私安全展示。保持单一 Go CLI、标准库实现和尽量少的配置项。

## 实施项

- [x] 将 GitHub 社区文件收纳到 `.github/`，不引入 `src/` 目录。
- [x] 采用最小标准 Go 布局：入口放入 `cmd/yuxin/`，应用代码和同包测试放入 `internal/app/`。
- [x] 新增 `yuxin update`，从 GitHub 最新正式 Release 下载当前平台的 CLI。
- [x] 新增 `yuxin update --force`；当前年份缺少节假日数据时每次启动提醒用普通 `yuxin update` 检查新版本，避免重复强制安装同一版本。
- [x] 下载后校验 Release 中的 SHA-256，再替换当前程序。
- [x] Release 产物使用稳定文件名，README 通过 `/releases/latest/download/...` 直达最新 ZIP。
- [x] 提供 Shell 和 PowerShell 安装脚本，在终端自动识别平台并安装 Latest Release。
- [x] 新增 `v` 视图快捷键，循环本地数据、固定合成演示和计算详情。
- [x] 退休只保留“年龄/出生年月 + 性别”自动估算，不收集身份证或参加工作年份。
- [x] 首次启动默认关闭退休和存款模块，首页聚焦今日入账和节假日倒计时。
- [x] 更新 README 看板、下载入口、命令和数据口径。
- [x] 运行格式化、单元测试、竞态测试、`go vet`、跨平台编译和 Release 产物校验。
- [x] CI 增加覆盖率门槛，全项目语句覆盖率不得低于 78%。
- [x] 完成后续路线验证，发布 `v0.3.0` 并由 GitHub 标记为 Latest Release。
- [x] 删除 `web` 命令和各平台桌面启动壳，产品只考虑终端用户。
- [x] 资产收口为一个存款金额，隐私收口为主界面三档循环快捷键。
- [x] 新增安全卸载命令，默认保留配置，`--purge` 才清除配置。
- [x] 用真实终端渲染内容生成宣传 MP4 和 README GIF，并提供可重复执行的生成脚本。

## 已确定的后续路线

### v0.5.0 质量提升

- [x] 节假日改为自动发现 `holidays-*.json`；Release ZIP 只附带最新年度文件，缺数据时不再提示强制重装同一版本。
- [x] Windows 自更新改为有限重试，失败后退出并清理暂存文件。
- [x] 退休倒计时收口为政策自动估算，不提供手动退休设置。
- [x] 存款预算统一使用“撑到退休”口径，并在面板标题显示响应式生活体感彩蛋。
- [x] 底部隐私和视图快捷键持续显示当前档位；非工作状态使用对应的轻量文案。
- [x] 增加真实 CLI 子进程、Unix 中断、Windows 更新/卸载脚本和四组 fuzz 测试。
- [x] 发布 Go 版本与 GitHub Actions 固定版本，Release Notes 从 CHANGELOG 生成，CI 增加 ShellCheck 与 actionlint。
- [x] 配置增加金额与工作天数上限、失败清理和可注入日期校验；新增 `doctor --strict`。
- [x] 节假日、退休、渲染格式和配置校验按职责拆到同包文件；长期工作日统计改为整周计算。
- [x] 完成本地体验验收，更新版本号和宣传素材，完成 v0.5.0 发布准备。

### 产品边界

- [x] 只维护终端仪表盘、单次快照和纯文本分享卡，不维护 Web 或桌面 GUI。
- [x] Release ZIP 只提供原生 CLI、默认配置、节假日数据、README 和 LICENSE。

### 纯文本分享卡片与图库

- [x] 新增一键生成固定宽度、纯文本、纯间距排版的可分享卡片。
- [x] 默认只使用合成数据；只有用户显式选择后才生成真实数据卡片，并在生成前提示隐私风险。
- [x] 提供完整功能演示卡，突出“一键生成可分享画面”。
- [x] 提供工作日倒计时卡，突出“无账号、离线运行、数据只在本地”。

### 退休倒计时

- [x] 年龄输入同时支持出生年月或完整日期并自动解析，输入 `0` 关闭。
- [x] 退休倒计时只支持年龄/出生年月与性别自动估算。
- [x] 按 18 岁起算退休进度，不要求工作起始年份。
- [x] 分三行同时显示整数剩余总年数、总月数和总天数，删除 `u` 单位切换。

### 数据与隐私

- [x] 增加配置导出和导入，在导出前明确标示可能含有的敏感字段。
- [x] 隐私模式与配置、退出同级，在仪表盘按 `p` 循环关闭、隐藏金额和存款、再同时隐藏退休三档。
- [x] 增加“一键清除本地数据”，并在删除前明确确认目标路径。

### 存款目标

- [x] 只配置一个“目标每月可花”，日、年口径由程序换算。
- [x] 口径为“如果现在停止工作，用存款撑到预计退休日期”；用进度条和“还差金额”展示，不引入寿命、收益率或通胀假设。

## 同类产品调研摘要

本轮调研只用于建立讨论清单，不自动扩大已确认的开发范围。

- [WTFUtil](https://github.com/wtfutil/wtf) 采用可配置模块、独立刷新频率、键盘焦点和主题，说明“按需开启模块”比把所有指标堆在默认首屏更易扩展。
- [Timewarrior](https://timewarrior.net/docs/) 提供日、周、月报告，标签、工作周、节假日、图表和数据导出，说明历史趋势与可导出数据是时间类产品的常见扩展方向。
- [Retirement Countdown](https://apps.apple.com/us/app/retirement-countdown/id1645215938) 提供全屏倒计时、天/周/月/秒单位、桌面小组件和分享卡片，说明“一键生成可分享画面”比让用户自行截取真实数据更安全。
- [RetirementTracker](https://play.google.com/store/apps/details?id=com.rententracker.app) 强调工作日倒计时、无账号、离线运行和本地数据，与 Yuxin 的本地优先方向一致。

## 暂不考虑

- Web、桌面 GUI、macOS 菜单栏、Windows 托盘、桌面小组件、主题和自定义模块排序。
- 本周/本月收入、发薪日、打点、加班、请假、历史趋势和提醒。
- 多地区节假日、系统日历和独立节假日更新。
- 银行同步、投资、收益预测和财务建议。

## 发布产物

GitHub Release 固定提供以下文件，以便最新版链接和自动更新器长期有效：

```text
yuxin-macos-arm64.zip
yuxin-macos-x86_64.zip
yuxin-windows-x86_64.zip
yuxin-linux-x86_64.zip
```

每个 ZIP 同时发布对应的 `.sha256` 文件。
