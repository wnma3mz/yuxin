# 余薪 Yuxin：终端 UI 方案

## 产品定位

Yuxin 是一个本地优先、启动即看的终端工作仪表盘。默认首屏只解决两个问题：

1. 今天已经赚了多少，还要多久下班。
2. 下一个节假日还有多久。

退休倒计时和存款是默认关闭的轻量可选项。程序不提供 Web 或桌面 GUI，也不连接银行、企业账户或其他在线数据源。

实时跳动的今日入账是第一视觉层级，工作进度与下班倒计时是第二层级，节假日进度是第三层级。退休倒计时和存款只有在用户主动开启后才出现。

颜色只承担状态含义：绿色表示正在工作或数据正常，黄色表示午休或即将下班，灰色表示休息日，红色只用于配置或数据错误。无颜色终端仍能完整阅读。

## 首次启动

首次启动不提问，自动写入默认 TOML 并进入仪表盘。默认值为：

- 月薪 8,000 元。
- 每月 22 个工作日。
- 周一至周五 09:00–18:00。
- 扣除 12:00–13:00 午休。
- 每秒刷新。
- 隐私模式、退休倒计时和存款关闭。

## 轻量配置

```text
1 今日入账   月薪 ¥8,000.00
2 工作时间   09:00–18:00，周 1,2,3,4,5
3 退休倒计时 关闭
4 存款       关闭
5 更多设置   刷新 1s
0 完成
```

- 退休按年龄/出生年月与性别自动估算；输入 `0` 关闭。
- 退休面板分别显示整数总年数、总月数、总天数，并保留从 18 岁起算的进度条。
- 工作时间中的午休只填写分钟数，输入 `0` 不扣除。
- 存款填写一个余额，并可选填写一个“目标每月可花”；目标用进度条和还差金额展示。
- 完整存款面板在“存款”标题后按日预算显示生活体感彩蛋；金额隐藏或界面进入紧凑布局时不显示。
- 隐私模式不在配置菜单内；仪表盘按 `p` 依次循环并保存“关闭 → 隐藏金额和存款 → 同时隐藏退休”。
- 刷新间隔和自定义口号放入“更多设置”。
- 进入配置项但没有改变内容时不写文件，也不提示“已保存”。

完整配置读写规则见 [config-review.md](config-review.md)。

## 隐私演示

按 `v` 循环“本地数据 → 合成演示 → 计算详情”三种画面。演示画面的标题持续显示“演示模式”，不读取本地工资、存款或退休配置；详情回到本地数据并遵守隐私档位。`yuxin share` 同样默认使用合成数据，只有显式传入 `--real` 才读取本地配置。

真实终端宣传素材已使用同一份固定数据生成 README GIF 和 MP4。`scripts/render-demo.sh` 会直接调用 Go 渲染函数获取看板和分享卡内容。

## 交互与命令

```text
yuxin                    启动自动刷新仪表盘
yuxin once               输出一次快照
yuxin config             修改本地配置
yuxin share              生成合成数据分享卡
yuxin doctor             检查配置和本地数据
yuxin update             安装 GitHub 最新正式版
yuxin uninstall          卸载程序并保留配置
yuxin uninstall --purge  卸载程序并清除配置
```

主屏快捷键：

```text
e 编辑配置   p 切换隐私档位   r 立即刷新
v 循环本地、演示和详情视图   q 安全退出
```

底部标签会随状态显示为“隐私·金额 / 隐私·全部”和“演示 / 详情”；空间不足时退回短标签，不使用容易闪烁的临时 Toast。

刷新间隔默认是 1 秒，可在配置中长期修改，也可用 `--interval` 临时覆盖。检测到非交互终端或输出被管道接收时，自动退化为 `once` 模式，避免 ANSI 控制符污染日志。

## 响应式终端

- `>= 100` 列：退休倒计时和存款可左右并列。
- `70–99` 列：可选项上下排列。
- `< 70` 列：使用紧凑单列，优先保留今日入账、下班倒计时和节假日。
- 终端高度不足时优先隐藏说明文字和次要内容，不隐藏今日入账与退出提示。

## 实现边界

项目使用 Go 标准库，不包含第三方模块或运行时依赖。TUI 使用 ANSI 刷新，配置使用一个本地 TOML 文件；业务计算与终端渲染分离，使持续刷新、单次快照和分享卡共用同一份模型。

不做 Web、桌面 GUI、银行同步、企业 SSO、投资收益预测或财务建议。
