# Web 更新记录

## 0.2.1 - 2026-07-19

- 已保存配置时可临时切换到演示数据，再一键回到我的数据；修改保存后自动回到实时个人视图，演示切换不会删除配置。
- 统一提高主页指标、公开图表、表单说明与移动端辅助文字的字号和对比度，改善浏览器及竖屏视频中的可读性。

## 0.2.0 - 2026-07-19

- 首屏右侧新增明确标记的动态演示余薪卡；演示工资按当前时间持续跳动，配置后原地切换为当前浏览器的实时数据。
- 本地配置使用版本化浏览器站点存储，关闭或刷新页面后保留；不生成设备 ID 或浏览器指纹，也不会自动带入匿名贡献。
- 浏览器端聚焦本地计算与匿名看板，不再提供纯文本分享选择；本地配置保存后可明确选择是否继续匿名贡献。
- 删除页面底部重复填写的匿名贡献长表单，改为复用本地配置的轻量弹窗；存款与退休字段默认不参与，必须单独勾选并再次确认。
- 浏览器退休配置支持“当前年龄 + 性别”本地估算或手动填写剩余年数；年龄和性别不会进入匿名贡献。
- 增加修改配置、清除本地数据、跨标签页同步、存储异常提示，以及桌面端和移动端适配。
- 增加本地配置校验、持久化、损坏数据恢复、工作状态、节假日、调休、退休估算和匿名字段选择测试。
- 页面标题更新为“余薪 · 摸鱼有数”，品牌副标题同步调整为 `YUXIN WEB`。
- 删除公开样本区重复的实时工资脉冲，实时入账只在首屏个人演示或本地数据中展示。
- 收紧宏观指标布局，补充导航当前位置、内容切换过渡、键盘焦点和移动端数据层级。
- 网页首次贡献时在当前浏览器保存随机编辑凭证；后续只更新原数值样本，数据库仅保存凭证摘要。

## 0.1.0 - 2026-07-18

- 新增公开匿名数据看板、本地区间比较和经审核的匿名回声；公开页面只读取聚合结果。
- 新增匿名贡献表单、单向提交确认和终端快捷入口。
- 匿名数值与回声分表保存，不记录应用层身份标识；当天数值延迟公开，并设置总样本与分桶隐私门槛。
- 删除首版看板不使用的年龄段和目标月支出字段，避免采集暂时无展示用途的数据。
- 数值与匿名回声改为独立 RPC 和独立事务；聚合数据按完整 10 份批次冻结发布，计数和中位数进一步降精度。
- 增加终端命令复制、回声换一批、图表区间定位、状态反馈和移动端适配。
- 重构看板叙事层级：公开样本、月薪和折算时薪合并为主卡；实时工资脉冲按本地当前时间、净工时中位数及随包节假日数据估算。
- 性价比大考与赛博躺平指数合并为同一切换舞台，象限和左右分布明细使用一致尺寸，桌面端可与宏观指标在一屏内阅读。
- 性价比大考采用「月薪 × 工时」四象限；赛博躺平指数采用「薪资 × 躺平压力」四象限，均以左上为更从容、右下为压力更高。
- 匿名回声改为三行对冲无缝跑马灯，增加「打气」类型，悬停暂停且遵循减少动效偏好。
- 浏览器 Tab 标题保持固定，不再切换为规避老板的提示文案。
- 增加响应格式校验、内容安全策略、Web CI、手动 GitHub Pages 工作流和进入 CI 的 Supabase 权限测试。
- 增加生产冷启动演示状态：首批 10 份真实公开样本形成前只展示明确标记的固定演示数据，达到门槛后原子切换为真实聚合结果。
- 首次部署至 [GitHub Pages](https://wnma3mz.github.io/yuxin/)，完成目标 Supabase、桌面端和移动端线上验收。
- 页面标题统一为“余薪 · 数据看板”，并将品牌色收敛为翡翠绿、琥珀黄和天空蓝的语义色板。
